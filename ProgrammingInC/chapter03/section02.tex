\section{运行我们的第一个程序}
{
    只要在命令行输入可执行文件的名字，就可以运行生成的可执行程序：

    \begin{codelist}
        \lstinputlisting[firstline = 4, lastline = 5]{./code/shell}
    \end{codelist}

    
    如果打算给最终生成的可执行文件另外起个名字，可以使用编译器的\emcode{\-o}选项，后面跟上想要的可执行文件的名字。
    例如：

    \begin{codelist}
        \lstinputlisting[firstline = 7, lastline = 9]{./code/shell}
    \end{codelist}
}

\endinput
